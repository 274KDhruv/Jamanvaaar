# Jamanwar Website

Design Direction:
Modern Gujarati luxury aesthetic.

Focus On:
- Gujarati culture
- authenticity
- premium typography
- warm hospitality
- elegant layouts
- subtle traditional motifs

Avoid:
- repetitive sections
- fake luxury language
- generic AI-generated layouts
- stock-looking restaurant design
- cluttered storytelling

Frontend Stack:
- Next.js
- Tailwind CSS

Rules:
- Use only two fonts
- Keep layouts clean
- Mobile-first responsiveness
- Maintain spacing consistency
- Use authentic Gujarati imagery
